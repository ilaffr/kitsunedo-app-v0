# Project Memory

## Core
Tech: Supabase (Auth, DB, Edge, Storage), Gemini AI (Flash, Pro Image).
Style: 'Ghost of Yōtei sumi-e' — pale washi bg, Noto Serif JP titles with thin underline, vermillion ginkgo seals, brush-stroke active states. Use `.washi-card`, `.yotei-title`, `.brush-active`, `.ginkgo-seal`. Avoid heavy 2px borders.
Layout: Pill-nav (mobile), sidebar (desktop). Kanji nav: 道, 学, 練, 績, 栄.
Rule: Never use animations on lesson pages (must remain static sumi-e).
Rule: Sanitize/encode Japanese characters in storage filenames.
Rule: Kitsune (ink-painted fox) mascot acts as guide via speech bubbles.
Rule: Daily Kitsune Tales must always be rooted in REAL Japanese culture/lore/history and include a "cultural_note" teaching the user the actual fact.
